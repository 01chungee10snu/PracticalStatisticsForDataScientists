# GitHub Pages 배포 가이드

## 🚀 GitHub Pages 활성화 방법

1. **GitHub 저장소 설정으로 이동**
   - 저장소 페이지에서 `Settings` 탭 클릭

2. **Pages 설정 찾기**
   - 왼쪽 메뉴에서 `Pages` 클릭

3. **Source 설정**
   - Source: `Deploy from a branch` 선택
   - Branch: `main` 선택
   - Folder: `/docs` 선택
   - `Save` 버튼 클릭

4. **배포 확인**
   - 몇 분 후 `https://your-username.github.io/PracticalStatisticsForDataScientists/` 에서 접근 가능
